Professional Self-Assessment

Throughout my journey in the Computer Science program at SNHU, I've transformed from someone who could write basic scripts into a software engineer capable of designing and implementing complex, production-ready systems. This capstone project—evolving a simple Arduino timer into a sophisticated Raspberry Pi-based astronomical light controller—perfectly encapsulates this growth.

Technical Foundation

My coursework has equipped me with fundamental skills that I've applied throughout this project. Data Structures (CS 260) taught me to think about efficiency and organization, which I applied when designing the circular buffer system for sensor data and implementing the LRU cache for astronomical calculations. Software Engineering (CS 310) provided the foundation for modular design and test-driven development, evident in my comprehensive test suite with 77 unit and integration tests. Database concepts from DAD 220 enabled me to design a normalized schema that transforms a stateless Arduino sketch into a data-driven platform capable of historical analysis.

Professional Growth

Beyond the technical skills, this program has shaped my professional values around writing maintainable, well-documented code. Early courses emphasized quick solutions, but later courses like Software Testing (CS 320) taught me that professional software requires comprehensive testing, clear documentation, and forward-thinking design. This shift in mindset is evident in my capstone, where I spent as much time writing tests and documentation as I did writing the actual functionality.

Collaboration & Communication

While this capstone was an individual project, my experience in collaborative environments throughout the program influenced its design. The modular architecture with clear interfaces between components reflects lessons learned from team projects about the importance of component independence.

Data Structures & Algorithms

Implemented advanced data structures including circular buffers for O(1) operations, priority queues for event processing, and trigonometric lookup tables with interpolation achieving 1000x performance improvements.

Software Engineering

Demonstrated professional practices through modular architecture, dependency injection, comprehensive error handling, and 96% test coverage across 77 automated tests.

Database Design

Created a six-table normalized schema with migration support, connection pooling, and thread-safe operations, transforming a stateless system into a data-driven platform.

Security Mindset

Implemented input validation, parameterized queries, rate limiting, and comprehensive error handling to prevent security vulnerabilities and information leakage.

Innovation

Applied event-driven architecture, predictive algorithms, and intelligent caching to create a system that learns from historical patterns and optimizes its behavior over time.

Portfolio Integration

The three enhancements in this portfolio—software design/engineering, algorithms/data structures, and databases—work together to demonstrate full-stack development capabilities. The modular Python architecture provides the framework for efficient astronomical calculations that are persisted and analyzed through the database layer. Each enhancement builds upon the others, creating a cohesive system that's greater than the sum of its parts.
